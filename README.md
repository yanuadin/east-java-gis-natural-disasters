## Installation

1. ```git clone https://github.com/yanuadin/east-java-gis-natural-disasters.git```
2. ```cd east-java-gis-natural-disasters```
3. ```cp .env.example .env```
4. buat database baru
5. isi credential database
6. ```composer install```
7. ```npm install && npm run dev```
8. ```php artisan serve```
